Tesouro Direto API version v1

http://localhost:8000/api/v1

/treasure-titles

All treasure titles available.

/treasure-titles get

get

get /treasure-titles

  • Request
  • Response

Query Parameters

  • action: (one of invest, redeem)

    Use the 'action' parameter to get a specific group of titles 'to invest' or 'to redeem', if the parameter is omitted all titles will be returned.

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • last_update: required(datetime-only)
  • invest: (array of TreasuryTitle)

    Items: TreasuryTitle

    • name: required(string)
    • annual_profitability: required(number)
    • minimum_investment: (number)
    • unit_price: required(number)
    • due_date: required(date-only)
  • redeem: (array of TreasuryTitle)

    Items: TreasuryTitle

    • name: required(string)
    • annual_profitability: required(number)
    • minimum_investment: (number)
    • unit_price: required(number)
    • due_date: required(date-only)
